Portfolio

Projects I've worked on

Intel Memory and Storage Tool

The Intel Memory and Storage Tool (IntelMAS) is a desktop application designed to manage SSD devices. It communicates with devices using the NVMe protocol to issue low-level commands, gather logs, and manage advanced settings.

Available in both CLI and GUI versions, the tool is written in C++ and interacts with system devices through OS layers. While primarily developed for Windows, it also runs on Linux. In my role, I developed device commands and log parsers by referencing NVMe documentation.

Amazon In-Store Mode

The "in-store mode" section of the Amazon app, use it to find deals at your local Whole Foods or Amazon Fresh and pay for your groceries using the in-store code.

I worked on designing and implementing the CI/CD and deployment architecture for this section of the application.

Amazon Dash Cart

Dash carts are smart shopping carts used in some Amazon Fresh stores. Fitted with a display in the handle to show deals and shopping lists to the user as they shop in the store.

I developed a system for delivering food item advertisements on the dash cart screen. My system fetches ad images from a company database and displays them in a GUI that I developed in javascript using Angular 10.

Amanda's Movie List

A movie watchlist web application that I developed to record my girlfriends ridiculously long list of movies she hasn't seen.

I created this as a side project to practice React development, it is built using the MERN stack (MongoDB, ExpressJS, React, NodeJS). The project includes a react frontend and a NodeJS REST API backend with a custom user authentication system, movie list NoSQL database and external api interactions.
